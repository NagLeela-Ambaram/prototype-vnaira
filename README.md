# prototype-vnaira
Contains all code base of Prototype for Vnaira 

## Installation (vnaira-web)

This is the web version of the prototype using Game Engine 

1. Clone this repository: `git clone https://github.com/NagLeela-Ambaram/prototype-vnaira.git`
2. Navigate to the client directory: `cd vnaira-web`
3. Install dependencies: `npm install`
4. Start the client: `npm run dev`
5. Access the client in your browser at: `http://localhost:3000`

## Installation (vnaira-r3f)

This is the web version of the prototype using React Three Fiber

1. Clone this repository: `git clone https://github.com/NagLeela-Ambaram/prototype-vnaira.git`
2. Navigate to the client directory: `cd vnaira-r3f`
3. Install dependencies: `npm install`
4. Start the client: `npm run dev`
5. Access the client in your browser at: `http://localhost:3001`

## Observations

What ever is built, is a small prototype of the idea. It wasnt enhanced much interms of UI/UX and other features.

- vnaira-web is the web version of the prototype using Game Engine
- vnaira-r3f is the web version of the prototype using React Three Fiber
- vnaira-web looks better than vnaira-r3f in terms of UI/UX
- vnaira-web is better in terms of performance 
- vnaira-web has more options to control the 3D environment from game engine.
- vnaira-r3f is better in terms Development tech stacks.
